# AdanPedia 

AdanPedia adalah aplikasi web e-commerce sederhana yang dibangun menggunakan Flask dan API publik DummyJSON.

## Fitur
- Daftar Produk
- Search Produk
- Filter Kategori
- Cart & Checkout
- Ringkasan Pesanan
- Simulasi Pembayaran

## Teknologi
- Python
- Flask
- DummyJSON API
- HTML, CSS, Bootstrap
